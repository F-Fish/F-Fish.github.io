等间隔取点

可使用 np.arange() 或 np.linspace()。

  • np.arange(start, stop, step)：类似于 range(start, stop, step)。
  • np.linspace(start, stop, num=50)：等间隔取 num 个点。

矩阵对角化

  • 定义矩阵（ndarray 类型）

第 m 行 n 列元素为 H[m][n]，从 0 起计数。

  • 使用 np.linalg (Linear Algebra) 对角化矩阵，返回矩阵本征值和本征矢量

对 Hermitian 矩阵，建议使用 np.linalg.eigh() 和 np.linalg.eigvalsh()。

如果要频繁使用该库，可简化：from numpy import linalg as LA。

Matplotlib.pyplot

散点图

使用 plt.scatter()。

  • 基本语法：plt.scatter(x, y)，x 和 y 为两个 float 或 array-like 对象。

输出：

欲保存图片，使用 plt.savefig()： plt.savefig("Fig 1.jpeg", format="jpeg")

  • 参数调节：
    • s：点的大小。
    • c：点的颜色。
    • marker：点的形状，见下表。
    • cmap：colormap，按数值给点赋不同颜色。常用的有 'viridis', 'plasma', 'inferno', 'magma', 'cividis'。
    • linewidths：点的边界线宽度。
    • edgecolors：点的边界线颜色。
