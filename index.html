Debye's Model of Solids

Posted on 2024-10-18

Debye 的固体

固体里面可以传递声波，这种声波就是原子在其平衡位置附近的小振动。如果这种振动的特征波长远大于原子间距，就可以把它看成是连续化的。

在一个各向同性的固体里面，声波可以有两种形式：一种是横波，一种是纵波。用 \(\boldsymbol\xi (\boldsymbol x,t)\) 来表示 \((\boldsymbol{x},t)\) 处原子相对平衡位置的移动，微观上的波动方程给我们

\[ \dfrac{\partial^2 \boldsymbol{\xi}}{\partial t^2}-c^2\nabla^2 \boldsymbol{\xi}=0 \]

纵波 \(c=c_l\)，横波 \(c=c_t\)。原子的一般运动可以写作

\[ \boldsymbol{\xi }\left( \boldsymbol{x},t \right) =\sum_{\boldsymbol{k}}{\boldsymbol{\xi }_l}\left( \boldsymbol{k} \right) \exp \left( i\boldsymbol{k}\cdot \boldsymbol{x}-i\omega_l t \right) +\sum_{\boldsymbol{k}}{\boldsymbol{\xi }_t}\left( \boldsymbol{k} \right) \exp \left( i\boldsymbol{k}\cdot \boldsymbol{x}-i\omega_t t \right) \]

其中

\[ \boldsymbol{\xi }_l\left( \boldsymbol{k} \right) =A_l\left( \boldsymbol{k} \right) \hat{\boldsymbol{e}}_{\boldsymbol{k}},\ \ \boldsymbol{\xi }_t\left( \boldsymbol{k} \right) =\hat{\boldsymbol{e}}_{\boldsymbol{k}}\times \boldsymbol{A}_t\left( \boldsymbol{k} \right) \]

这一步操作，相当于把我们从实空间变到动量空间。我们预期量子化后的哈密顿量会形如下式：

\[ \mathcal{H} =\hbar \left( \sum_{\boldsymbol{k}}{\left( a_{\boldsymbol{k}}^{l\dagger}a_{\boldsymbol{k}}^{l}+\frac{1}{2} \right) c_l\left| \boldsymbol{k} \right|}+\sum_{\boldsymbol{k}}{\left( a_{\boldsymbol{k}}^{t\dagger}a_{\boldsymbol{k}}^{t}+\frac{1}{2} \right) c_t\left| \boldsymbol{k} \right|} \right) \]

固体的能谱相应为

\[ E\left( \{ n_{\boldsymbol{k}}^{l},n_{\boldsymbol{k}}^{t} \} \right) =\hbar \left( \sum_{\boldsymbol{k}}{\left( n_{\boldsymbol{k}}^{l}+\frac{1}{2} \right) c_l\left| \boldsymbol{k} \right|}+\sum_{\boldsymbol{k}}{\left( n_{\boldsymbol{k}}^{t}+\frac{1}{2} \right) c_t\left| \boldsymbol{k} \right|} \right) \]

这里的 \(n_{\boldsymbol{k}}^{l},n_{\boldsymbol{k}}^{t}=0,1,2,\ldots\) 可诠释为准粒子的粒子数。

因此，配分函数按定义写为

\[ \mathcal{Z} =\sum_s{\exp \left( -\beta E_s \right) =\sum_{\{ n_{\boldsymbol{k}}^{l},n_{\boldsymbol{k}}^{t} \}}{\exp \left[ -\beta \hbar \left( \sum_{\boldsymbol{k}}{\left( n_{\boldsymbol{k}}^{l}+\frac{1}{2} \right) c_l\left| \boldsymbol{k} \right|}+\sum_{\boldsymbol{k}}{\left( n_{\boldsymbol{k}}^{t}+\frac{1}{2} \right) c_t\left| \boldsymbol{k} \right|} \right) \right]}} \]

接下来是最关键的一步：观察上式，我们注意到，在我们做对配分函数求和的时候，各个 \(n_{\boldsymbol{k }}^{l}\) 和 \(n_{\boldsymbol{k }}^{t}\) 都是独立的。因此，配分函数里面对 \(\{ n_{\boldsymbol{k}}^{l},n_{\boldsymbol{k}}^{t} \}\) 的求和，也可以相应拆成对各个 \(n_{\boldsymbol{k }}^{l}\) 和 \(n_{\boldsymbol{k }}^{t}\) 独立求和后再相乘的结果：

\[ \mathcal{Z} =\prod_{\boldsymbol{k}}{\{ \sum_{n_{\boldsymbol{k}}^{l}=0}^{\infty}{\exp \left[ -\beta \hbar \left( n_{\boldsymbol{k}}^{l}+\frac{1}{2} \right) c_l\left| \boldsymbol{k} \right| \right]}\cdot \sum_{n_{\boldsymbol{k}}^{t}=0}^{\infty}{\exp \left[ -\beta \hbar \left( n_{\boldsymbol{k}}^{t}+\frac{1}{2} \right) c_t\left| \boldsymbol{k} \right| \right]} \}} \]

这里的每一个求和，都可以被认为是一个“近独立子系”的配分函数 \(z_{\boldsymbol{k}}\)。这种子系并不是真实的一个粒子实体，而是一种独立的传播模式 \(\boldsymbol{k}^{(l)}\) 或 \(\boldsymbol{k}^{(t)}\)。它们之所以可以被视为近独立子系，本质上是因为在配分函数里它们没有耦合在一起。值得指出的是，这里的每个近独立子系并不是相同的，它们对应各种不同波矢的不同传播方式（横波/纵波）。因此，我们通常熟知的公式（\(\mathcal{Z}=z^N\)）是不可以使用的，因为在这个公式的推导过程中，暗示了每个近独立子系的配分函数完全相同。

每一项独立求和的结果为

\[ \sum_{n_{\boldsymbol{k}}^{l}=0}^{\infty}{\exp \left[ -\beta \hbar \left( n_{\boldsymbol{k}}^{l}+\frac{1}{2} \right) c_l\left| \boldsymbol{k} \right| \right]}=\frac{\exp \left( -\beta \hbar c_l\left| \boldsymbol{k} \right|/2 \right)}{1-\exp \left( -\beta \hbar c_l\left| \boldsymbol{k} \right| \right)} \]

自由能，作为配分函数的负对数，相应可以写成

\[ F=-kT\ln \mathcal{Z} =\sum_{\boldsymbol{k}}{\frac{\hbar c_l\left| \boldsymbol{k} \right|}{2}}+\sum_{\boldsymbol{k}}{\frac{\hbar c_t\left| \boldsymbol{k} \right|}{2}}+kT\sum_{\boldsymbol{k}}{\ln \left( 1-\exp \left( -\beta \hbar c_l\left| \boldsymbol{k} \right| \right) \right)}+kT\sum_{\boldsymbol{k}}{\ln \left( 1-\exp \left( -\beta \hbar c_t\left| \boldsymbol{k} \right| \right) \right)} \]

而且，原先对 \(\boldsymbol{k}\) 的求和可以连续化为积分：

\[ \sum_{\boldsymbol{k}}{}=\alpha V\int_{\boldsymbol{k}}{\frac{d^3\boldsymbol{k}}{\left( 2\pi \right) ^3}} = \alpha V\int_{\boldsymbol{k}}{\frac{4\pi k^2 dk}{\left( 2\pi \right) ^3}} \]

对于横波，\(\alpha=2\)；对于纵波，\(\alpha=1\)。

要一直把横波和纵波两部分分开写的话，形式上显得格外复杂。为了简化，我们把自由能的常数部分（也就是零点能）扔掉不管，后面两项中，在第一个求和号里代换 \(\omega=c_l |\boldsymbol{k}|\)，在第二个求和号里代换 \(\omega=c_t |\boldsymbol{k}|\)，这样我们就把两项求和给合成一项了：

\[ F = kT\sum_{\boldsymbol{\omega }}{\ln \left( 1-\exp \left( -\beta \hbar \omega \right) \right)} = kT\int_{\omega}{\frac{V\omega ^2\mathrm{d}\omega}{2\pi ^2}\left( \frac{1}{u_l^3}+\frac{2}{u_t^3} \right) \ln \left( 1-\exp \left( -\beta \hbar \omega \right) \right)} \]

接下来的讨论就是熟知的了。
